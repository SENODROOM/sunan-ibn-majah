# Contributing

## Getting Started

```bash
git clone https://github.com/SENODROOM/sunan-ibn-majah.git
cd sunan-ibn-majah
```

## Making Changes

- JS changes → edit files in `src/`
- Python changes → edit files in `python/sunan_ibn_majah/`
- Data changes → edit `data/majah.json` then run `node scripts/build.mjs`

## Publishing

Publishing is automatic via GitHub Actions on every GitHub Release.
