# CLI Reference — sunan-ibn-majah

```
majah <id>                     Hadith by ID
majah <id> -a                  Arabic only
majah <id> -b                  Arabic + English
majah <chapter> <n>            nth hadith of a chapter
majah --search "<query>"       Search (top 5)
majah --search "<query>" --all Show all results
majah --random                 Random hadith
majah --chapter <id>           All hadiths in chapter
majah --chapters               List all chapters
majah --info                   Book metadata
majah --react                  Generate useMajah React hook
majah --version
majah --help
```
