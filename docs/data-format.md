# Data Format — majah.json

```json
{
  "metadata": {
    "id": 1, "length": 4341,
    "arabic":  { "title": "سنن ابن ماجه", "author": "الإمام محمد بن يزيد ابن ماجه" },
    "english": { "title": "Sunan Ibn Majah", "author": "Imam Muhammad ibn Yazid Ibn Majah" }
  },
  "chapters": [ { "id": 1, "arabic": "...", "english": "The Book of Purification" } ],
  "hadiths": [
    { "id": 1, "chapterId": 1, "arabic": "...", "english": { "narrator": "...", "text": "..." } }
  ]
}
```

`data/chapters/` is generated by `node scripts/build.mjs`. Do not edit manually.
